# Svelte Contact Book Site

This is a simple contact book made in Svelte.

![Add Contact Page](/attachments/svelte-contact-book-site-add-contact-page.png)

Contacts can be created from the '/add-contact' page pictured above. Contacts are stored in a local.db file on the server.

## Usage

Visit [svelte-contact-book-site.onrender.com](https://svelte-contact-book-site.onrender.com)
